# Cyberia Channel
aka cyberia-chan

Intro
------
Cyberia Channel is a Websocket based online IM platform based on drrr-like-chat (Express JS)

Usage
-----

1.	Use `npm i` to install.
2.	Run with `npm start`
3.	Open `http://your-ip:3000`
